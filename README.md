# New-Art-Commerce-and-Science-College-Website

### 1. File Structure
Ensure all files are placed in the correct directories as shown in the structure.

### 2. Running the Website

#### Option 1: Using Live Server (Recommended)
1. Install VS Code
2. Install "Live Server" extension
3. Right-click on `index.html` and select "Open with Live Server"

### 2. Customization

- **Colors**: Modify CSS variables in `css/main.css`:
```css
  :root {
      --primary-color: #1e40af;
      --secondary-color: #7c3aed;
      --accent-color: #f59e0b;
  }
```

- **Content**: Edit individual page files in the `pages/` directory.

- **Components**: Modify header, footer, etc. in the `includes/` directory.

### 3. Browser Compatibility

- Modern browsers (Chrome, Firefox, Edge, Safari)
- IE11 not supported (uses modern JavaScript features)
